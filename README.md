# First yahoo
Open source tools that directly probe the target can be used to gather information about it. Examining the target from an adversarial standpoint and determining the amount of information that can be obtained is helpful when testing for vulnerabilities. Kali offers numerous tools for collecting data. I'll use Dimitri to identify the public internet registration information of the yahoo.com website and Port scanning on a virtual machine on my network.
